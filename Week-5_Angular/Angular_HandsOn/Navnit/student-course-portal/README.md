# Student Course Portal (Angular 20)

Digital Nurture 5.0 — Week 5 (Angular) / Week 6 (Angular + GIT/CI/CD)  
Angular Hands-On Exercises 1–10 in one project.

## Setup

```bash
npm install
npm install -g json-server
```

## Run mock API

```bash
npm run api
```

JSON Server serves `db.json` on http://localhost:3000

## Run app

```bash
npm start
```

Open http://localhost:4200

## Build & test

```bash
npm run build
npm test
```
